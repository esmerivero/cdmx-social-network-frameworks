# Rehaciendo mi red social usando librerías y frameworks

En este proyecto se hará uso de la librería de ReactJS, permitiendo ahorrar tiempo en la configuración.
El objetivo principal del proyecto es tener un primer acercamiento a la librería, profundizando su funcionamiento,
entender los ciclos de vida y comprender la interacción de los componentes.

A continuación se dejará una lista de los alcances esperados.
